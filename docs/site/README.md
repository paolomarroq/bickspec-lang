# BickSpec documentation site

This folder contains the canonical offline BickSpec documentation site.

Main page:

```text
bickspec_documentation.html
```

Assets and screenshots are kept locally with relative paths so the page remains portable and works when opened directly from disk.

The VS Code extension ships a packaged runtime copy under:

```text
vscode-extension/media/docs/
```

When updating the canonical documentation in `docs/site/`, refresh the packaged extension copy as well.
